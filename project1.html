Secure Home Automation

Executive Summary

My college capstone project, "Secure Home Automation," was a collaborative effort with three classmates. This sophisticated setup integrated a wide array of sensors, such as vibration sensors for windows, ultrasonic sensors for lighting, RFID900 for door access, and temperature sensors for climate control.



Utilized Raspbian OS and various components including

  • Accelerometer Sensor MPU6050
  • Piezo speaker
  • ESP32
  • RFID Sensor
  • Ultrasonic Sensor
  • Temperature Sensor

Developed an advanced system capable of

  • Detecting glass breakage
  • Integrated a wide array of sensors such as:
    • Vibration sensors for windows
    • Ultrasonic sensors for lighting
    • RFID900 for door access
    • Temperature sensors for climate control

Created a bespoke Android application with features including:

  • Card management
  • Door control
  • Lighting adjustments
  • Temperature regulation
  • Window testing

Showcased proficiency in:

  • Smart systems
  • Sensor integration
  • User-centric design
  • Custom PCB design
  • Integration of IoT devices
  • Security protocols and encryption methods
